#### Description
The Full-width CTA paragraph is a simple way to call attention to important information. It does not have an image and spans the full width of the page when there is not a rail navigation. Many of its fields are optional for flexibility. This story will create the semantic markup and styles needed for the paragraphâ€™s display.

#### Variables

eyebrow: short text / optional
title: title / required
desc: short text / optional
cta: 
  url: url / optional
  title: short text / optional
~~~
